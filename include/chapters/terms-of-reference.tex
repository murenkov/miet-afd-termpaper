\chapter{Техническое задание}

Спроектировать и рассчитать АФАР со следующими параметрами:
\begin{itemize}
    \item АФАР должна быть настроена на центральную частоту $F_c = 6~\text{ГГц}$ ($\lambda = 5~\text{см}$);
    \item коэффициент усиления должен быть не менее, чем $K_p = 18~\text{дБ}$;
    \item коэффициент шума не должен превышать значение $K_{ш} = 3.5~\text{дБ}$;
    \item количество разрядов равно $p = 4$;
    \item углы сканирования по осям $x$ и $y$ равны $\Theta_{\text{ск} x} = \pm15^\circ$ и $\Theta_{\text{ск} y} = \pm30^\circ$ соответственно;
    \item ширины лучей по уровню $0.5~\text{дБ}$ по осям $x$ и $y$ равны $\Theta_{0.5 x} = 3^\circ$ и $\Theta_{0.5 y} = 6^\circ$ соответственно;
    \item уровень боковых лепестков $t$ не превышает $-15~\text{дБ}$.
\end{itemize}

Требуется рассчитать $\Pi_\text{прм}$, $N$, $N_x$, $N_y$, $d_x$, $d_y$, $\delta\Theta_x$, $\delta\Theta_y$, $\Delta$, а также нарисовать схему возбуждения, выбрать тип излучателя и рассчитать его характеристики.
